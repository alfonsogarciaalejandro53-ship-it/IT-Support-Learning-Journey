# Router Basics
# What is a Router?
A router connects different networks and forwards packets between them
## Gateway
The Gateway is the router's IP adress in the local network.
## Routing 
Is the process of choosing the path for a packet
## Routing table 
A routing table contains the routes used to decide where packet should be forwarded
## Default Route
0.0.0.0/0
Used when no more specific routes exists.
